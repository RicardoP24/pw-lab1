Programação Web

Na unidade curricular de Programação Web abordaremos princípios básicos associados ao desenvolvimento para a web, tanto do lado Back-end como no lado front-end

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • PYTHON
  • DJANGO
  • JAVASCRIPT

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal de entrega obrigatória, de aplicação da matéria lecionada
